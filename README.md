# Projeto Agenda de Contatos

## Objetivo Geral

Criar um aplicativo de contatos que permita armazenar os dados em um banco de dados

## Objetivo Especificos:

* Inserir
* Atualizar
* Remover
* Deletar tudo
* Importa
* Exportar

## Tecnologias Usadas

- Linguagen: Python
- Sqllite
- PyQt

## Instalação

pip -m venv venv

source venv/bin/activate

pip install -r requirements.txt


## Execução

python agenda.py
